# Smiley ![](https://imgur.com/FDwWkMf)
An offline Electron app for using emojis in Office apps, chat apps and websites.

## Usage
* Pick an emoji. 
* The app will copy your emoji to your clipboard.
* Paste the emoji wherever you want.
